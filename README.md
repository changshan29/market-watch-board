# 盯盘资讯看板

一个智能的财经资讯聚合平台，实时追踪市场动态。

## 功能特点

- 📊 多源聚合：整合主流财经平台资讯
- 🎯 智能分类：AI 自动识别行业和主题
- 🔍 精准筛选：按来源、主题、A股概念快速定位
- 🚀 实时更新：持续追踪最新市场动态
- 📱 响应式设计：支持桌面和移动端访问

## 技术栈

- 后端：Node.js
- 数据处理：Python 3
- AI 分类：Claude Haiku
- 部署：Railway / Render

## 快速开始

### 本地运行

```bash
# 安装依赖
pip3 install -r requirements.txt

# 启动服务
node server.js

# 访问
open http://localhost:3220
```
